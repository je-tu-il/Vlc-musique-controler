# Vlc-musique-controler
VLC Overlay Controller : Prenez le contrôle de votre musique sans encombrer votre écran. Une interface overlay élégante pour VLC qui permet de gérer la progression, la vitesse et le volume à la molette, et qui disparaît automatiquement quand vous ne l'utilisez pas.


Une interface légère, élégante et non-intrusive pour contrôler VLC lorsqu'il tourne en arrière-plan.

Fonctionnalités Principales
  Affichage Intelligent : L'interface apparaît automatiquement à chaque changement de musique ou lors de la mise en pause, puis disparaît pour laisser place à votre contenu.
  Mode "Focus" : L'overlay reste affiché tant que votre curseur le survole, empêchant sa disparition automatique pendant que vous l'utilisez.

Navigation & Contrôle :
  Barre de progression : Déplacez-vous n'importe où dans le morceau d'un simple clic.
  Volume à la molette : Ajustez le volume du système intuitivement en scrollant sur l'interface.
  Vitesse de lecture : Menu déroulant pour adapter la vitesse (idéal pour podcasts et livres audio).

Informations Visuelles :
  Affichage de la Pochette (Cover Art) en haut à gauche.
  Indication précise du Time Code et de la durée totale.

Ergonomie :
  Raccourci Global : Invoquez l'interface à tout moment avec Alt + é (configurable).
  Positionnement Mémoire : Déplacez la fenêtre où vous voulez sur l'écran ; elle mémorisera sa position lors de la prochaine réapparition.
